# Sales Analytics Dashboard

This project analyzes a global retail dataset to uncover business insights and build an interactive Power BI dashboard.

## Tools Used

* Python (Pandas, NumPy, Matplotlib)
* Jupyter Notebook
* Power BI
* Git / GitHub

## Project Workflow

1. Data cleaning and preprocessing
2. Feature engineering
3. Exploratory data analysis
4. Business insights extraction
5. Dashboard development in Power BI

## Dataset

Global Superstore dataset with **51,000+ orders** across multiple regions.

## Key Insights

* Technology generates the highest revenue.
* Consumer segment drives most sales.
* Some regions generate revenue but very low profit.
* Sales show strong seasonal patterns with Q4 peaks.

## Repository Structure

analysis – cleaned dataset for Power BI
data – raw dataset
notebooks – Python analysis notebook
dashboard – Power BI dashboard files
